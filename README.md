## 使用方法（PyQt版）

1. ``git clone https://github.com/Colin130716/yay-plus_PyQt.git``
2. ``cd <git clone到的路径>/yay-plus_PyQt/src``
3. ``python3 -m pip install -r requirements.txt``
4. ``chmod 700 askpass.py``
5. ``chmod +x askpass.py``
6. ``./askpass.py --init``
7. ``python3 home.py``
